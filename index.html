Find the right Domain name

Domain name should follow these basic rules:
- Brainstorm 5 top keywords
- make the domain unique
- choose the .com domain if available
- make it easy to type and remember
- keep the name short
- create and fulfill the expectations (e.g.: careebuilder.com)
- set yourself apart with a brand
- no hyphens and numbers

URL strategy

URLs should follow these basic rules:
- Describe the content
- Keep it short
- Static, it should not change
- Descriptive is better than numbers
- Use keywords
- Few folders
- Avoid use of subdomains if possible
- Hyphens separate best
- Don't be case-sensitive
- Don't append extraneous data

Images Alt and Title

Image Alt Each image published, also those that could be defined as irrelevant should have a description that includes keywords and a full description of what is in the picture. This information will constitute the “Alternative to image text”, it will be the text seen in the case the browser won’t load the image.

Example. For a traveling-focused web site the following image could be described in many ways:

Poor Optimal
Tour Eiffel The magnificent Tour Eiffel in the middle of the Paris skyline, where you can spend romantic moments. Description of each image should be provided to developer in order to make sure it is included in the HTML code.

Image title This parameter should provide additional information and follow the rules of a regular title: it should be relevant, short, catchy, and concise. IN some browsers it pops up when you hover over an image. For the example above we could have the following image title:
 Title
 Tour Eiffel
Title Paris most visited monument

Internal Links

Internal links make the site structure.
Site structure should:
- not have more than 3 or 4 levels of links
- all the pages have to be achievable in few clicks
- links should be clear

External Link : TITLE

Title parameter offers advisory information about the element for which it is set. Visual browsers frequently display the title as a "tool tip". Do not duplicate the anchor text as it could be seen as a keyword stuffing.

External Link : ANCHOR TEXT

Should use keywords related to the content of the page itself, not the destination page, but the page where the link is located.
- Relevance: the links for a page comes from a site that talks about the same topic.
- Authority: obtain links for "hubs" related to specific topics. The more relevant hubs the higher will be the authority.
- Trust: How strong are the intentions of the editors of the published pages to not being a spammer? This intention provides the reliability of a connection to another web site.

The best is to use text and not images if possible. Images are not yet indexable per the content itself.
